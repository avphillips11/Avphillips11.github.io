Case Study — Enterprise Security

SkyNOPE — Endpoint AI Security Platform

AI coding assistants run with user credentials, execute Bash commands, read sensitive files, and call APIs, but the enterprise security stack was never designed to see any of it. I identified this gap from inside a managed detection and response environment and built the solution through AI orchestration, without writing a single line of code by hand. SkyNOPE fuses two independent endpoint witnesses (EDR + Claude Code hooks) into one session timeline. When the witnesses disagree, that disagreement is the security signal.

Case Study — GRC Program Management

Project TOG — AI-Powered GRC Team

As the primary owner of a full GRC and Privacy program covering SOC 2, ISO 27001/27701, PCI DSS, NIST 800-171, GDPR, AI governance, TPRM, and a risk program revamp, with a team of two, I needed a way to scale capacity without scaling headcount. I built a structured AI team of 12 specialists inside Claude Code, each with a defined role, domain expertise, and persona. The result: a cross-framework unified controls library, a comprehensive risk program revamp plan, a full AI governance strategy, and a live GRC command center. All produced in under three weeks.
